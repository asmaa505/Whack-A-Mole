

# Whack-A-Star Game 🎮

**Whack-A-Star** is a simple interactive web game where the player must click on stars appearing in random boxes to score points before the timer runs out.

The project is built using:  
- **HTML5** – for the basic structure of the game  
- **CSS3 + Tailwind CSS** – for a stylish and responsive design  
- **JavaScript** – to handle star movement, scoring, and the game timer  

---

## 🕹️ How to Play

1. A star appears in a random box every 1.5 seconds.  
2. Clicking on the star turns it into smaller stars briefly and increases your score by 1.  
3. The timer starts from 30 seconds.  
4. When the time ends, you are redirected to a final score page.  
5. Press **Replay** to play again.

---

## 💻 How to Run Locally

1. Download the project folder (HTML, CSS, JS, Images).  
2. Open `game.html` in any modern browser (Chrome, Edge, Firefox).  
3. Enjoy playing!

---

## 🌟 Features

- Fully **responsive design** for all screen sizes.  
- **Animated stars** appear inside fixed-size boxes.  
- **Live scoreboard** updates automatically.  
- Final score page with **Replay** button.
